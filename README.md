## Sobre o projeto
Esse projeto teve por objetivo criar um modelo de _machine learning_ com o uso do algoritmo XGBoost para prever churn em uma empresa de telecomunicações. Para isso, foi realizada uma análise exploratória dos dados, onde se conheceu melhor o data set e foi detectado o desbalanceamento da classe alvo, a ausência de outliers e de dados ausentes. Então, criaram-se 7 modelos diferents de previsão de _churn_ onde foram avaliados de acordo com o valor de _Recall_. Com um estudo para se detectar os melhores valores para os hiperparâmetros do XBGBoost, foi possível aprimorar o modelo de forma que ele teve o maior valor de _Recall_ entre os modelos criados. Foi realizado ainda um trabalho de _feature engineering_ com a criação de 4 novos atributos.
